Database

All our data is kept in firestore realtime NoSQL database and therefore the application needs to be constantly connected to the internet.

Changes made in the database are therefore instantly observed within our application. The application also stores all its images within the Cloudimage which will ensure the best optimisation and fast delivery of the images.for more information about cloudimage visit.

Security

All users will need to be authenticated before they can access the app either through email and password or google signin. Firebase Authentication uses the data to enable end-user authentication, and facilitate end-user account management. It also uses user-agent strings and IP addresses to provide added security and prevent abuse during sign-up and authentication. Managers will have to surbsribe to the application which will ensure only authentic individuals are given access to the system data. for more information on firebase security visit here
